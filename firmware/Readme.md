# Firmware Arduino

Se eligió un Arduino Pro Micro (versión USB-C) ya que física,ente encaja muy bien en el dispositivo, tieme pines suficientes para el proyecto y da la opción de comunicación HID, aunque dadas las limitaciones del protocolo HID finalmente se desestimó manteniendo un protocolo serie más rápido y flexible.

El Arduino Pro Micro se programa en Arduino IDE usando la opción de placa **Arduino Leonardo**. No usar las opciónes  *Arduino Pro or Pro mini* ni *Arduino Micro*: no son compatibles.
